"Nominatim" for Baseball Data

Nominatim is OpenStreetMap’s open-source search service: type anything, get a precise entity back. Baseball API applies that philosophy to the diamond. You can look up “trout”, “BOS 1912”, or a Retrosheet game ID and we map it to canonical players, teams, parks, or seasons—then expose the underlying data through consistent endpoints. It’s a lightweight layer on top of Retrosheet play-by-play, FanGraphs park factors and constants, Lahman (via SABR), and the MLB Stats API, so you can prototype tools, analyses, or visualizations without rebuilding the ingestion stack.
